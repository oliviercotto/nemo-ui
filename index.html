Nemo-ui

Webpage allowing to generate init files for Nemo the forward-time individual-based quantitative genetics simulation software.

View Nemo-ui on GitHub

Overview

This static web page allows to specify the parameters defining a Nemo simulation and download the corresponding files.

The configuration file (or init file), is a text file with one parameter per line in a key/value scheme where the key is the parameter name, and the value its argument value
